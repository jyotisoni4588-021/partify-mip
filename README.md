# Partify – Part Time Job Finder Web System

Partify is a Flask-based web application developed as a Minor Project (MIP).  
It helps students and local job seekers find nearby part-time jobs, and allows local businesses to post job openings easily.

---

## Project Links
Local App (after running):
http://127.0.0.1:5000

GitHub Repository:
https://github.com/jyotisoni4588-021/partify-mip

---

## Features
- User signup and login
- Post part-time job listings
- Browse available jobs
- Filter jobs by category (Tuition, Cafe, Mall, Operator, Other)
- View complete job details
- Google Maps directions to job location
- My Jobs section to edit or delete jobs

---

## Tech Stack
- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Tools: VS Code, Google Maps
